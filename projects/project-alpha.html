Cryoablation Planning Benchmark

Dataset benchmark for cryoablation planning

Overview

This project introduces a dataset benchmark to evaluate algorithms for cryoablation planning, enabling consistent comparison of methods for image‑guided intervention planning.

Goals

  • Provide curated, high‑quality imaging data relevant to cryoablation planning.
  • Define standardized evaluation protocols and metrics.
  • Enable reproducible comparison across planning algorithms.
